# 📦 Guia de Instalação\n\nGuia completo para instalar e configurar o Check-up de TI.\n\n## ⚡ Instalação Rápida (2 minutos)\n\n### Windows 10/11\n\n1. **Clone o repositório**\n   ```bash\n   git clone https://github.com/seu-usuario/check-up-ti.git\n   cd check-up-ti\n   ```\n\n2. **Execute o script**\n   ```powershell\n   .\\executar.bat\n   ```\n   Ou apenas **clique duas vezes** em `executar.bat`\n\n3. **Verifique o relatório**\n   - Abra `C:\\Users\\[seu-usuario]\\Desktop\\Relatorio_Rede.html` no navegador\n\n## 📋 Requisitos de Sistema\n\n### Mínimos\n- **SO:** Windows 7 SP1+\n- **PowerShell:** 3.0+\n- **RAM:** 10 MB\n- **Espaço em disco:** 1 MB\n\n### Recomendados\n- **SO:** Windows 10/11\n- **PowerShell:** 5.1+ (PowerShell Desktop) ou 7.0+ (PowerShell Core)\n- **Navegador:** Edge, Chrome ou Firefox (para visualizar HTML)\n\n## 🔧 Instalação Manual\n\n### Opção 1: Via Git\n\n```bash\n# Instale Git se não tiver (https://git-scm.com)\ngit clone https://github.com/seu-usuario/check-up-ti.git\ncd check-up-ti\npowershell .\\Check-upTI.ps1\n```\n\n### Opção 2: Download ZIP\n\n1. Visite: https://github.com/seu-usuario/check-up-ti/archive/main.zip\n2. Extraia o arquivo\n3. Abra a pasta\n4. Clique duplo em `executar.bat`\n\n### Opção 3: PowerShell Direto\n\n```powershell\n# Altere a política de execução (se necessário)\nSet-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser -Force\n\n# Execute o script\n.\\Check-upTI.ps1\n```\n\n## 🚀 Primeiro Uso\n\n### Configuração Inicial\n\nO script já vem com configuração padrão. Para personalizar:\n\n1. **Edite `Check-upTI.ps1`** (se usar editor de texto)\n2. **Localize a seção \"CONFIGURAÇÕES\"** (linhas ~20-25)\n3. **Modifique os IPs conforme sua rede:**\n   ```powershell\n   $IP_IMPRESSORA = \"192.168.1.50\"  # Seu IP aqui\n   ```\n\n### Teste Inicial\n\n```powershell\n# No PowerShell, execute:\nTest-Connection 192.168.1.1 -Count 1\n```\n\nSe receber resposta, o script funcionará!\n\n## 🛠️ Configuração Avançada\n\n### Múltiplos Perfis\n\nCrie cópias do script com nomes diferentes:\n\n```\nCheck-upTI-Setor1.ps1\nCheck-upTI-Setor2.ps1\nCheck-upTI-Backup.ps1\n```\n\nCada um pode testar IPs diferentes.\n\n### Integração com Agendador de Tarefas\n\n1. **Abra Agendador de Tarefas** (Win + R → `taskschd.msc`)\n2. **Criar Tarefa Básica**\n3. **Configurar:**\n   - Nome: \"Check-up de TI\"\n   - Acionador: Diariamente às 9:00 AM\n   - Ação: Iniciar programa\n   - Programa: `powershell.exe`\n   - Argumentos: `-ExecutionPolicy Bypass -File \"C:\\caminho\\Check-upTI.ps1\"`\n\n## ✅ Verificação de Instalação\n\n### Teste de Sintaxe\n\n```powershell\n# Verifique se há erros de sintaxe\n[System.Management.Automation.PSParser]::Tokenize(\n    (Get-Content 'Check-upTI.ps1'), [ref]$null\n)\n```\n\n### Teste Funcional\n\n```powershell\n# Execute o script\n.\\Check-upTI.ps1\n\n# Verifique se o arquivo foi criado\nTest-Path \"$env:USERPROFILE\\Desktop\\Relatorio_Rede.html\"\n```\n\n### Resultado Esperado\n\n```\nTrue  #
