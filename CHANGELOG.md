# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-22

### Adicionado
- ✍️ Versão inicial do script Check-upTI
- 🔍 Testes de conectividade para Roteador, Internet e Impressora
- 📊 Geração de relatório em HTML com status visual
- 📑 Documentação completa do script em formato PowerShell Help
- 📤 README.md com instruções detalhadas de uso
- 🔖 .gitignore configurado para o projeto
- 📜 LICENSE MIT para uso livre
- 🔠 executar.bat para facilitar execução (clique duplo)
- 📋 CHANGELOG.md para rastrear versões

### Características da Versão 1.0.0
- ✅ Interface limpa e intuitiva
- ✅ Cores diferenciadas (verde para ONLINE, vermelho para OFFLINE)
- ✅ Suporte a personalização de IPs testados
- ✅ Relatório salvo automaticamente na Desktop
- ✅ Nome do usuário logado no relatório
- ✅ Código bem documentado e profissional
- ✅ Sem dependências externas
- ✅ Execução rápida e leve
